## 수정사항

### Front
1. 상품목록 조회
- 등록일 삭제 [o]
- 등록일 자리에 상품 상세 보여주기 [o]
- 구매 완료되면 목록에서 삭제 [o]
- 상품번호, 상품가격 검색 삭제 [o]



2. 구매목록 조회
- "NO" 대신에 구매 상품 이름 보여주기 [o]
- 판매중일때 "배송정보 수정"에 수정 버튼 나오게하기 [o]

3. 상품 상세조회
- '수정' 버튼 삭제 [o]


### back
1. 상품목록 조회
-  ProductDAO getList 판매중 상태만 가져오게 수정 [o]
-  searchCondition 삭제 [o]


2. 구매목록 조회
-  searchCondition 삭제 [o]
